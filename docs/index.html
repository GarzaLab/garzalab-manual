1 Welcome

Welcome to the Neuroimaging and Psychiatry Lab aka GarzaLab. This is a walkthrough for new students and staff, as well as the place to find FAQs and the code of conduct. Hopefuly this manual with provide you with an idea of how the lab functions, provide clarity on my expectations on lab members and what you can expect from me.

All lab members will be expected to read this lab manual. The manual is a dynamic document, so some sections will be added and/or revised as the lab grows and develops.

This manual was mostly inspired by the great Grange-Lab manual. This manual is licence under a CC Attribution 4.0 Licence.

Let me know if you have any questions.

Kind regards
